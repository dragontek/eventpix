Welcome to EventPix! This guide helps hosts and guests get the most out of their shared photo experience.

## For Hosts üé§

### Creating an Event
1.  **Sign In**: Log in as a host using your email or social account (Google/Apple).
2.  **Create**: Click **+ New Event** on your dashboard.
3.  **Details**: Fill in the event name and a unique **Event Code** (e.g., `WEDDING2025`).
    -   **Description**: Add a welcome message or details.
    -   **Dates**: Set the start and end times.
    -   **Visibility**: Choose *Public* (searchable), *Unlisted* (share link only), or *Private*.
    -   **Join Mode**: 
        -   *Open*: Anyone with the code can join.
        -   *PIN*: Users need a 4-digit PIN to enter.
        -   *Invite Only*: Restricted access.

### Managing Your Event
-   **Dashboard**: View all your created events.
-   **Edit**: As an owner, you can edit event details at any time from the event page.
-   **Moderation**: You can bypass approval queues for your own photos.

### Live Slideshow
-   **Launch**: From the event page, click the **Play** icon to start a full-screen slideshow.
-   **Real-time**: New photos appear automatically as guests upload them. Perfect for projecting on a big screen at the venue!

## For Guests ü§≥

### Joining an Event
-   **Enter Code**: On the home page, enter the **Event Code** provided by the host.
-   **Search**: Use the **Search Public Events** button to find public gatherings by name.
-   **QR Code**: Scan the event's QR code to join instantly.

### Uploading Photos
1.  **Camera**: Tap the **Camera** icon to snap a photo directly in the app.
2.  **Gallery**: Tap the **Upload** icon to select multiple photos from your device's library.
3.  **Contribute**: Your photos are instantly added to the shared gallery!

### Viewing the Gallery
-   **Real-time Feed**: Watch the gallery grow as other guests upload.
-   **Download**: Tap any photo to view it full screen and download it.
